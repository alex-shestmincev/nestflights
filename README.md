## Description
This service is created for getting flights

## Installation

```bash
$ nvm use
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```

Go to [http://localhost:3000/flights](http://localhost:3000/flights) to see the flights

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
